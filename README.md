# 🎨 Single Color Memory Game

**Can you remember a color perfectly?**

You’ll see a random color for a few seconds… then it disappears.
Your challenge is to **recreate it from memory** using Hue, Saturation, and Brightness controls.

---

## 🎮 How It Works

* 👀 View a color (3–5 sec)
* 🧠 Memorize its look
* 🎛 Adjust HSB sliders to match it
* 📊 Get a score (0–100%) based on accuracy

---

## 🏆 Why It’s Fun

* Tests your visual memory
* Improves color perception
* Quick, addictive gameplay

---

## ⚡ Goal

Get as close to **100% accuracy** as possible.

**Sounds easy… but it’s not.**
